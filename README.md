 # Windows-server-domin-lab
  Hands- on lab documenting the setup of a windows Server virtual machine, promotiing to a Domain Dontroller, and basic Active Directory user and DNS management. 

 ## Lab Environment 
 - Windows Server 2025
 - Virtual Machine (Azure)
 - Active Directory
 - Active Directory Domain Servers
 - DNS


  ## The Lab Objectives
  - Install Windows Sever
  - Configure Active Directory Domain Services
  - Promote server to Domain Controller
  - Create users and organizational units
  - Document the process using GitHub
 
  ## Tools Used 
  - Windows Server 2025
  - Active Directory Domain Services
  - DNS
